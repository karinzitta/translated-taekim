Prática com Conjugações de Verbos no Negativo
----------------------------------------

Aprendemos como classificar os seguintes verbos no exercício de prática anterior. Agora vamos colocar esse conhecimento em prática conjugando os mesmos verbos para o negativo, dependendo do tipo de verbo. A primeira resposta foi fornecida como exemplo.

| verbo | negativo |
|---|---|
| 行く | 行かない |
| 出る | 出ない |
| する | しない |
| 買う | 買わない |
| 売る | 売らない |
| 食べる | 食べない |
| 入る | 入らない |
| 来る | こない |
| 飲む | 飲まない |
| しゃべる | しゃべらない |
| 見る | 見ない |
| 切る | 切らない |
| 帰る | 帰らない |
| 書く | 書かない |
