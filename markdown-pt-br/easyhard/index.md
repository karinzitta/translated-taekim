Esta é uma lição curta e fácil sobre como transformar verbos em adjetivos que descrevem se aquela ação é fácil ou difícil de fazer. Basicamente, consiste em mudar o verbo para o radical e adicionar 「やすい」 para fácil e 「にくい」 para difícil. O resultado então se torna um i-adjetivo regular. Bem fácil, né?

Usando 「～やすい、～にくい」 para descrever ações fáceis e difíceis

*   Para descrever uma ação como sendo fácil, mude o verbo para o radical e adicione 「やすい」. Para descrever uma ação como sendo difícil, anexe 「にくい」 ao radical.
    Exemplos

    1.  食べ_る_ → 食べ_やすい_
    2.  しゃべ_る_ → しゃべ_り_ → しゃべり_にくい_

O resultado se torna um i-adjetivo regular.

| | Positivo | Negativo |
|---|---|---|
| Não-passado | 食べにく_い_ | 食べにく_くない_ |
| Passado | 食べにく_かった_ | 食べにく_くなかった_ |

### Exemplos

1.  この字は_読みにくい_
    Esta caligrafia é difícil de ler.
2.  カクテルはビールより_飲みやすい_。
    Coquetéis são mais fáceis de beber do que cerveja.
3.  部屋が暗かったので、_見にくかった_。
    Como o quarto estava escuro, era difícil de ver.

Como observação: Cuidado com 「見にくい」 porque 「醜い」 é um adjetivo raramente usado que significa "feio". Imagino se é só coincidência que "difícil de ver" e "feio" soam exatamente igual.

É claro que você sempre pode usar alguma outra estrutura gramatical que já aprendemos para expressar a mesma coisa usando adjetivos apropriados como 「難しい」、「易しい」、「簡単」、「容易」 etc. As duas frases seguintes são essencialmente idênticas em significado.

1.  その肉は_食べにくい_。
    Essa carne é difícil de comer.
2.  その肉を_食べるのは難しい_。
    A coisa de comer essa carne é difícil.

Variações de 「～にくい」 com 「～がたい」 e 「～づらい」

### Vocabulário

6.  とても – muito

14.  待ち合わせ 【ま・ち・あわ・せ】 – combinação de encontro
15.  分かる 【わ・かる】 (u-verb) – entender
16.  場所 【ば・しょ】 – local

O kanji de 「にくい」 na verdade vem de 「難い」, que também pode ser lido como 「かたい」. Por isso, você também pode adicionar uma versão sonorizada 「～がたい」 como sufixo verbal para expressar a mesma coisa que 「にくい」. 「にくい」 é mais comum na fala enquanto 「がたい」 é mais adequado ao meio escrito. 「にくい」 tende a ser usado para ações físicas, enquanto 「がたい」 geralmente é reservado para ações menos físicas que não exigem movimento real. No entanto, parece não haver uma regra rígida sobre qual é mais apropriado para um dado verbo, então sugiro pesquisar ambas as versões no Google para verificar a popularidade de uma combinação específica. Você também deve sempre escrever o sufixo em hiragana para evitar ambiguidades na leitura.

### Exemplos

1.  彼との_忘れがたい_思い出を大切にしている。
    Estou tratando com carinho as memórias difíceis de esquecer com ele.
2.  とても_信じがたい_話だが、本当に起こったらしい。
    É uma história muito difícil de acreditar, mas parece (por boato) que realmente aconteceu.

Ainda outra variação mais grosseira de radical + 「にくい」 é usar 「づらい」 em vez disso, que é uma versão ligeiramente modificada de 「辛い」（つらい）. Não confunda com o mesmo 「辛い」（からい）, que significa picante!

### Exemplos

1.  日本語は_読みづらい_な.
    Cara, o japonês é difícil de ler.
2.  待ち合わせは、_分かりづらい_場所にしないでね。
    Por favor, não escolha um local difícil de encontrar para o encontro combinado.
