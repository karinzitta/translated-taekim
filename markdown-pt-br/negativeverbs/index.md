Agora que vimos como declarar coisas e realizar ações com verbos, queremos ser capazes de expressar o negativo. Em outras palavras, queremos dizer que tal e tal ação _não_ foi realizada. Isso é feito conjugando o verbo para a forma negativa, assim como o estado de ser para substantivos e adjetivos. No entanto, as regras são um pouco mais complicadas.

Conjugando verbos para o negativo
-----------------------------------

Agora faremos uso das classificações de verbos que aprendemos na última seção para definir as regras de conjugação. Mas antes, precisamos cobrir uma exceção muito importante às regras de conjugação negativa: 「ある」.

*   ある (u-verbo) – existir (inanimado)
*   いる (ru-verbo) – existir (animado)

「ある」 é um u-verbo usado para expressar a existência de objetos inanimados. O verbo equivalente para objetos animados (como pessoas ou animais) é 「いる」, que é um ru-verbo normal. Por exemplo, se você quisesse dizer que uma cadeira está no quarto, usaria o verbo 「ある」, mas se quisesse dizer que uma _pessoa_ está no quarto, precisaria usar o verbo 「いる」. Esses dois verbos 「ある」 e 「いる」 são bem diferentes de todos os outros verbos porque descrevem existência e não são ações reais. Você também precisa ter cuidado para escolher o correto com base em objetos animados ou inanimados.

Bem, a razão pela qual menciono isso aqui é que o negativo de 「ある」 é 「ない」 (o que significa que algo não existe). As regras de conjugação para todos os outros verbos estão listadas abaixo, junto com uma lista de verbos de exemplo e suas formas negativas.

＊ = exceções particulares a esta conjugação

Regras de conjugação para verbos negativos

*   **Para ru-verbos:** Remova o 「る」 e adicione 「ない」
    Exemplo: 食べる + ない = 食べ_ない_
*   **＊Para u-verbos que terminam em 「う」:** Substitua 「う」 por 「わ」 e adicione 「ない」
    Exemplo: 買う + わ + ない = 買_わない_
*   **Para todos os outros u-verbos:** Substitua o som de vogal "u" pelo equivalente de vogal "a" e adicione 「ない」
    Exemplo: 待つ + た = 待_たない_
*   **Exceções:**
    1.  する → しない
    2.  くる → こない
    3.  ＊ある → ない

Exemplos de conjugação para a forma negativa

| ru-verbo | u-verbo | exceção |
|---|---|---|
| 見る →　見ない | 話す →　話さない | する → しない |
| 食べる →　食べない | 聞く →　聞かない | くる → こない |
| 寝る →　寝ない | 泳ぐ →　泳がない | ＊ある → _ない_ |
| 起きる →　起きない | 遊ぶ →　遊ばない | |
| 考える →　考えない | 待つ →　待たない | |
| 教える →　教えない | 飲む →　飲まない | |
| 出る →　出ない | ＊買う →　買_わ_ない | |
| 着る →　着ない | 帰る →　帰らない | |
| いる →　いない | 死ぬ →　死なない | |

### Exemplos

Aqui estão os exemplos de frases da última seção conjugados para a forma negativa.

1.  アリスは_食べない_。
    Quanto à Alice, não come.
2.  ジムが_遊ばない_。
    Jim é o que não brinca.
3.  ボブも_しない_。
    Bob também não faz.
4.  お金が_ない_。
    Não há dinheiro. (lit: O dinheiro é a coisa que não existe.)
5.  私は_買わない_。
    Quanto a mim, não compro.
6.  猫は_いない_。
    Não há gato. (lit: Quanto ao gato, não existe.)
