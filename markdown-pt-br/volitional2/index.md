Aprendemos em uma lição anterior que a forma volitiva é usada quando alguém se dispõe a fazer algo. Nesta seção, vamos abordar outras formas em que a forma volitiva é usada, principalmente a forma volitiva negativa.

Talvez você já tenha visto a forma volitiva negativa em uma tabela de conjugação de verbos e se perguntado: "Para que serve isso?" A resposta é: para pouca coisa, ou seja, para ser mais preciso, há várias formas em que ela pode ser usada, mas quase todas são extremamente rígidas e formais. Na verdade, é tão rara que encontrei apenas [uma explicação](http://www.csse.monash.edu.au/~jwb/wwwverbinf.html) em inglês na web ou em qualquer outro lugar. (Também encontrei esta [aqui](http://www.geocities.jp/niwasaburoo/32kanyuuisi.html#32.10) em japonês.)

O volitivo negativo é usado para expressar intenção negativa. Isso significa que há uma vontade de que algo **não** aconteça ou que alguém se dispõe a **não** fazer algo. Como resultado, porque alguém está tentando não fazer algo, provavelmente não vai acontecer. Essencialmente, é uma versão muito rígida e formal de 「でしょう」 e 「だろう」. Embora esta forma praticamente nunca seja usada em conversas do dia a dia, você ainda pode ouvi-la em filmes, etc.

Os verbos são conjugados para o volitivo negativo simplesmente acrescentando 「まい」 ao verbo. Outra alternativa é acrescentar 「まい」 ao radical. A conjugação do volitivo negativo é bem diferente daquelas com as quais estamos acostumados porque é sempre a última conjugação a ser aplicada, mesmo para a forma masu. Não há como conjugar 「まい」 para a forma masu; simplesmente se acrescenta 「まい」 à conjugação na forma masu.

Usando 「まい」 para expressar a vontade de não fazer algo

*   **Para ru-verbos:** Acrescente 「まい」 ao verbo ou ao radical do verbo.
    Exemplo 1:　見る → 見る_まい_
    Exemplo 2:　見る → 見_まい_
*   **Para u-verbos:** Acrescente 「まい」 ao final do verbo
    Exemplo: 行く_まい_
*   **Exceções:**
    1.  する → するまい ou しまい
    2.  くる → くるまい
*   Esta conjugação deve sempre vir por último. Para a forma masu, acrescente 「まい」 ao verbo na forma masu.
    Exemplo: な_る_ → な_り_ → なり_ます_ → なります_まい_

### Exemplos

1.  相手は剣の達人だ。そう簡単には_勝てまい_。
    Seu oponente é um mestre da espada. Duvido que você consiga vencer tão facilmente.
2.  そんな無茶な手段は_認めますまい_！
    Não aprovarei um método tão irrazoável!

Já aprendemos que você pode usar a forma volitiva para dizer "vamos" e para expressar uma tentativa de fazer algo. Mas isso não significa que você pode usar o volitivo negativo para dizer "vamos não" ou "tentar não". O tom desta gramática é de determinação muito firme em não fazer algo, como você pode ver nos exemplos a seguir.

1.  その時までは決して彼に_会うまい_と心に決めていた。
    Até aquele momento, havia decidido em meu coração não o encontrar de jeito nenhum.
2.  あの人は、二度と嘘を_つくまい_と誓ったのです。
    Aquela pessoa havia jurado nunca mais mentir.

Para expressar "vamos não", você pode usar o verbo 「やめる」 com o volitivo regular. Para expressar um esforço em não fazer algo, você pode usar 「ようにする」 com o verbo negativo.

1.  明日に行くのを_やめよう_。
    Vamos não ir amanhã. (lit: Vamos desistir de ir amanhã.)
2.  肉を食べない_ようにしている_。
    Estou tentando não comer carne.

Aprenderemos agora uma gramática que é de fato prática para uso cotidiano, usando a gramática do volitivo negativo. Basicamente, podemos usar tanto as formas volitiva quanto a volitiva negativa para dizer que não importa se algo vai acontecer ou não. Isso é feito acrescentando 「が」 tanto à forma volitiva quanto à forma volitiva negativa do verbo que não importa.

Usando o volitivo para expressar ausência de relação

*   Acrescente 「が」 às formas volitiva e volitiva negativa do verbo.
    Exemplos

    1.  食べる → 食べ_よう_、食べ_まい_ → 食べよう_が_食べまい_が_
    2.  行く → 行_こう_、行く_まい_ → 行こう_が_行くまい_が_

### Exemplos

1.  あいつが大学に_入ろうが入るまい_が、俺とは関係ないよ。
    Se aquele cara vai entrar na faculdade ou não, não tem nada a ver comigo.
2.  時間が_あろうがあるまい_が、間に合わせるしかない。
    Haja tempo ou não, não há nada a fazer a não ser chegar a tempo.
3.  最近のウィルスは強力で、プログラムを_実行しようがしまいが_、ページを見るだけで感染するらしい。
    Os vírus ultimamente estão poderosos e, quer você execute um programa ou não, ouvi dizer que se espalham só de ver a página.

Já descobrimos que o volitivo negativo pode ser usado como uma espécie de versão formal de 「でしょう」 e 「だろう」. Você pode se perguntar: como fazer o mesmo para o volitivo? A resposta é conjugar o verbo 「ある」 do estado-de-ser formal 「である」 para o volitivo, produzindo 「であろう」. Lembre-se de que 「でしょう」 já pode ser usado como uma forma polida, então esta forma está ainda um degrau acima em termos de formalidade. Veremos que tipo de linguagem usa esta forma nos exemplos.

Usando 「であろう」 para expressar probabilidade

*   Acrescente 「であろう」 ao substantivo, adjetivo ou verbo.
    Exemplos

    1.  困難 → 困難_であろう_
    2.  する → する_であろう_

### Exemplos

1.  今後50年、人間が直面_するであろう_問題に正面から向き合って、自ら解決をはかりつつ、そのノウハウが次の産業となるシナリオを考えたい。(de [www.jkokuryo.com](http://www.jkokuryo.com/papers/online/20020107.htm))
    Gostaria de enfrentar diretamente os problemas que os seres humanos _provavelmente encontrarão_ nos próximos 50 anos e, enquanto traço soluções, usar esse conhecimento para pensar em cenários que se tornarão a próxima indústria.
2.  もちろん、生徒数減少の現在、学科の新設は_困難であろう_が、職業科の統廃合や科内コースの改編などで時代に合わせた変革が求められているはずである。(de [www1.normanet.ne.jp](http://www1.normanet.ne.jp/~ww100114/library/li-01.htm))
    Claro que criar novos cursos de estudo _provavelmente será difícil_ neste período de redução no número de alunos, mas com reorganizações de estudos profissionais e cursos dentro das disciplinas, supostamente há demanda por reformas adequadas para esta era.

Aprendemos na lição sobre gramática formal que 「ではない」 era o negativo de 「である」. Então como diríamos algo como 「であろう」 mas para o negativo? A resposta é usar outro tipo de volitivo para negativos e adjetivos i, usado apenas em contextos formais e literários. Você pode pensar nessa gramática como uma versão muito antiquada para adjetivos i e terminações negativas em 「い」.

A regra de conjugação é simples: remova o último 「い」 e acrescente 「かろう」. Você pode usá-la para negativos e adjetivos i, assim como a forma de passado 「かった」.

Usando 「かろう」 para expressar o volitivo para terminações em 「い」

*   Remova o último 「い」 e acrescente 「かろう」
    Exemplos

    1.  ではな_い_ → ではな_かろう_
    2.  早_い_ → 早_かろう_

### Exemplos

1.  どんな商品でもネットで販売するだけで売上が伸びるというもの_ではなかろう_。
    Não é necessariamente o caso que as vendas aumentam só por vender qualquer tipo de produto na internet.
2.  運動を始めるのが_早かろう_が_遅かろう_が、健康にいいというのは変わりません。
    Seja cedo ou tarde para começar a se exercitar, o fato de que é bom para a saúde não muda.
3.  休日で_あろう_が、_なかろう_が、この仕事では関係ないみたい。
    Seja feriado ou não, parece que não faz diferença para este trabalho.
